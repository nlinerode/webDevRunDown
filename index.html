Books (Well just one...)

Next, I'll put a very simple book here.

HTML and CSS: Design and Build Websites

This book was designed for beginners and hobbyists, but I read through it in a month (shocker right?) and it helped solidify what I already knew a LOT. It has lots of good knowledge to get you going on the basics of web development. There is also a Javascript book by the same guy, but I haven't really gotten into yet (it's much more complicated). It's not too expensive, (maybe even at the library) and it's a pretty comprehensive starter guide.

I have more books I recently got but haven't read yet, so that's the only book I can recommend so far.

Back to the web!

There are MANY many learning sites. Some are paid, subscription, and free. I'm cheap of course so I find the best free ones I can.

Ok I'll start a list here.

  1. Traversy Media

    Since I already showed you his video, Traversy Media has some great content on his channel on Youtube. Absolutely wonderful. It's pretty fast paced, but it's chock full of good stuff.

    He has a video on creating a responsive web page (https://www.youtube.com/watch?v=Wm6CUkswsNw&t=2167s) which turned my existing knowledge (and from the book above) into a working exercise. It was greatly helpful into helping utilize my skills.

  2. FreeCodeCamp

    So I stumbled on this a few months ago. Didn't take to it right away, but then I fell in love with it. Got hooked. It's basic, but you get to code right away and do lots and lots of trial and error.

    The downside,(upside maybe?) is that it isn't very "teachy". As in, you'll have to research how to do pretty much every exercise and challenge that you find. But as you study more web dev, you'll realize quickly that there is no complete answer guide to anything. You'll have to find it yourself.

    But, it is cool. You make real projects and a portfolio for yourself. You learn a bit along the way, through the site and through self-teaching and research.

    You also have to set up a GitHub account which is handy and helpful. Last weekend I learned how to properly use it and it's awesome.

    Also, you can network! Through it I've found a group of people in Charlotte that are starting to meet up weekly. It's a necessity to coding and I'm sure there's something up that way too.

  3. Udemy

    I just recently started using this site. There were a bunch of free courses on here and I snagged them.

    However! There is one course in particular that I picked up on sale. I guess it goes on sale for $10-$20 very often, so you should be able to get it. He's a great teacher and it's very well explained. I wish I would've found it sooner as many others have said.

    Colt Steele's "The Web Developer Bootcamp"

    Definitely look into that and keep an eye on it. It's worth getting.

  4. W3Schools

    So W3C is the standard and the people who define HTML and all the goodness there. This is their learning, hands-on site. I've heard people talk smack about it, but I love it. I taught myself Bootstrap just by going through their tutorial on it. I also am going through CSS3 on the site too. It's a great reference if anything.

    I do recommend it highly though. It's pretty straighforward, lots of reference material, and lots of examples you can see, and even edit on the site. That way, you can experiment and play with it.

  5. MDN

    Mozilla Developer Network. I just recently found this (through the bootcamp course above I just mentioned) and have been using it since. It's a great comprehensive site with lots of details. It pretty much as all the web standards on it and examples etc.

  6. Google Web Devs

    Can't say much about this one because I haven't really used it, but they do offer some free courses through it.

  7. HTML Dog

    Another one I haven't used, but I've heard good about. Similar to W3Schools if I'm correct.

  8. Others

    There is a plethora of others, and I'm sure I can send you more. They're out there. There are others like Treehouse and CodeAcademy that are popular but paid courses.

Text Editors!

Almost forgot! Text editors.

So you can use Notepad (or whatever Linux's equivalent is). I'm actually not sure what Linux has, but Sublime Text is great. It's quite nice and it's techincally free to use forever. (I'll pay for it eventually). A good text-editor will help coding when you start. Notepad is nice to get the feel for syntax and muscle memory and general memorization. Then work into an IDE, more formal text editor.
